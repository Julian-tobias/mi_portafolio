Sistema de Béisbol

Proyecto de panel estadístico para equipos de béisbol, con enfoque en organización clara de datos y navegación rápida. Es una muestra de cómo presentar información deportiva de manera ordenada y profesional.

Volver al portafolio Ver código

Características

  • Presentación de estadísticas por equipo.
  • Interfaz limpia y fácil de leer.
  • Enfoque en datos deportivos y análisis visual.

Cómo mostrarlo

Este proyecto se puede dejar en una carpeta separada dentro del repositorio para que el cliente acceda directamente desde GitHub Pages. Eso permite ver el demo desde una URL específica.
